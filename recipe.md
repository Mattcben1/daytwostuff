## Recipe for getting into a new environment and using codespace
#### 1. Fork the repo you intend to use, or create one
#### 2. From there, navigate to the green code button in your repo, click codespaces, and follow it to allow opening of a codespace
#### 3. You'll be prompted to the VS web browser, but it's important that you access it locally --> File -> New Window -> Connect to Codespaces...You should have ready to be accessable.
#### 4. Ensure you download the neccessaey packages (Ipykernal, python env., jupyter) if you need them
#### 5 Create and activate a virtual environment and install all local dependencies (requirements.txt, etc.)
